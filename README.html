Problemen Oplossen

Onbekende Bestandsformaten

Als een bestand niet automatisch wordt herkend, navigeert de applicatie naar het tabblad "File Mapping" waar u handmatig een mapping kunt maken.
1 Custom File Mapping Maken
In het "File Mapping" tabblad kunt u:
  • Kolommen uit het bronbestand slepen naar de template-velden
  • Vaste waarden instellen door op een template-veld te klikken
  • Auto-mapping gebruiken voor automatische suggesties
  • De mapping opslaan voor toekomstig gebruik
  • Keywords toewijzen voor automatische herkenning

Templates en Mappings Delen met Collega's

üîÑ JSON Export/Import Systeem: Deel eenvoudig uw templates en file mappings met teamleden via JSON bestanden.

Templates Delen

1 Template Exporteren
In het "Templates" tabblad:
  • Selecteer het template dat u wilt delen
  • Klik op "Export Template" om een JSON bestand te downloaden
  • Deel dit JSON bestand met uw collega's via email of gedeelde mappen
2 Template Importeren
Uw collega's kunnen het template gebruiken door:
  • Naar het "Templates" tabblad te gaan
  • Op "Import Template" te klikken
  • Het JSON bestand te selecteren dat u hebt gedeeld

File Mappings Delen

1 Mapping Exporteren
In het "File Mapping" tabblad:
  • Maak een file mapping voor een specifiek bestandsformaat
  • Klik op "Save Broker Template" om automatisch een JSON backup te downloaden
  • Het bestand bevat zowel de mapping als de parsing configuratie
2 Mapping Importeren
Collega's kunnen de mapping gebruiken door:
  • Naar het "File Mapping" tabblad te gaan
  • Op "Import Template JSON" te klikken
  • Het gedeelde JSON bestand te selecteren
  • Een keyword toe te voegen voor automatische herkenning
üí° Teamwork Tip: Maak een gedeelde netwerkmap waar het hele team JSON bestanden kan opslaan en ophalen. Zo heeft iedereen toegang tot dezelfde templates en mappings.

Automatische Keyword Herkenning

Na het importeren van een file mapping kunnen uw collega's:
  • Een keyword toevoegen (bijvoorbeeld "DeVerkoper" of "ClientNaam")
  • Bestanden die dit keyword in de naam bevatten worden automatisch verwerkt
  • Geen handmatige mapping meer nodig voor toekomstige bestanden!

Geavanceerde Functies

Template Beheer

In het "Templates" tabblad kunt u:
  • Nieuwe templates maken of bestaande bewerken
  • Templates importeren/exporteren als JSON
  • Kolommen toevoegen, bewerken of verwijderen

File Mapping Opslaan

Wanneer u een mapping heeft gemaakt in het "File Mapping" tabblad, kunt u deze opslaan als herbruikbaar template voor toekomstige bestanden van hetzelfde formaat.

Zoek- en Filterfunctionaliteit

In het Results tabblad kunt u real-time zoeken in alle getoonde records om specifieke gegevens snel te vinden.

Tips voor Optimaal Gebruik

Keywords: Voeg betekenisvolle keywords toe aan uw file mappings. Bestanden met deze keywords in de naam worden automatisch verwerkt.
Batch Verwerking: U kunt meerdere bestanden tegelijk uploaden. De applicatie verwerkt ze sequentieel en navigeert automatisch naar het juiste tabblad.
Download Locatie: Stel eenmalig uw voorkeursmap in via Instellingen. Alle downloads openen dan automatisch in deze map.
Data Validatie: Controleer altijd de preview voordat u data downloadt om er zeker van te zijn dat de mapping correct is.

Technische Vereisten

  • Browser: Moderne browser met JavaScript ingeschakeld
  • Bestanden: Excel (.xlsx) (toekomst: PDF bestanden)
  • Internet: Niet vereist na eerste laden (offline gebruik mogelijk)
Privacynota: Alle gegevens worden lokaal in uw browser verwerkt. Er worden geen bestanden naar externe servers verzonden.
